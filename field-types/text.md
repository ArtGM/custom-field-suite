## Text

Generates a single-line text field

### Field Options

| Option | Purpose |
|--------|---------|
| Default Value | A default value, which appears before first save |

### Return Value

(string) The inputted text

### Template Usage

{% highlight php %}
<?php
echo CFS()->get( 'first_name' );
{% endhighlight %}
